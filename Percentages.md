# PERCENTAGES: Complete Mastery Guide for Government Exams

## 1. FOUNDATION TO ADVANCED COVERAGE

### **BASIC CONCEPTS (Foundation Level)**

#### **A. What is a Percentage?**

**Definition:** Percentage means "per hundred" or "out of 100."

**Symbol:** % 

**Basic Idea:**
```
50% = 50 out of 100 = 50/100 = 1/2 = 0.5
```

**Real-world Examples:**
- **Discount:** 20% off on ‚Çπ500 ‚Üí Save ‚Çπ100
- **Marks:** 75% in exam ‚Üí 75 marks out of 100
- **Tax:** 18% GST on ‚Çπ1000 ‚Üí Pay ‚Çπ180 extra
- **Interest:** 10% return on ‚Çπ10,000 ‚Üí Earn ‚Çπ1,000

---

#### **B. Basic Conversions**

**1. Fraction to Percentage:**
```
Multiply by 100

Example: 3/4 = (3/4) √ó 100 = 75%
```

**2. Percentage to Fraction:**
```
Divide by 100

Example: 25% = 25/100 = 1/4
```

**3. Decimal to Percentage:**
```
Multiply by 100

Example: 0.65 = 0.65 √ó 100 = 65%
```

**4. Percentage to Decimal:**
```
Divide by 100

Example: 40% = 40/100 = 0.4
```

---

#### **C. Common Percentage-Fraction-Decimal Equivalents (MEMORIZE!)**

**Essential Values:**
```
50% = 1/2 = 0.5
25% = 1/4 = 0.25
75% = 3/4 = 0.75
33.33% = 1/3 = 0.333...
66.67% = 2/3 = 0.667...
20% = 1/5 = 0.2
40% = 2/5 = 0.4
60% = 3/5 = 0.6
80% = 4/5 = 0.8
10% = 1/10 = 0.1
12.5% = 1/8 = 0.125
16.67% = 1/6 = 0.1667
```

**Why memorize?** Instant conversion saves 10-15 seconds per problem!

---

#### **D. Finding Percentage of a Number**

**Formula:**
```
x% of N = (x/100) √ó N
```

**Example:** 30% of 500
```
= (30/100) √ó 500
= 0.3 √ó 500
= 150
```

**Shortcut:** "Of" means multiply!

---

#### **E. Finding What Percent One Number is of Another**

**Formula:**
```
(Part/Whole) √ó 100 = Percentage
```

**Example:** 40 is what % of 200?
```
= (40/200) √ó 100
= 0.2 √ó 100
= 20%
```

---

#### **F. Finding the Whole When Percentage is Known**

**Formula:**
```
Whole = (Part √ó 100) / Percentage
```

**Example:** 30% of what number is 60?
```
Whole = (60 √ó 100) / 30 = 200
```

### **INTERMEDIATE LEVEL TECHNIQUES**

#### **A. Percentage Increase/Decrease**

**Percentage Increase:**
```
% Increase = [(New Value - Original) / Original] √ó 100
```

**Example:** Price increases from ‚Çπ500 to ‚Çπ600
```
% Increase = [(600-500)/500] √ó 100 = (100/500) √ó 100 = 20%
```

---

**Percentage Decrease:**
```
% Decrease = [(Original - New Value) / Original] √ó 100
```

**Example:** Price decreases from ‚Çπ800 to ‚Çπ640
```
% Decrease = [(800-640)/800] √ó 100 = (160/800) √ó 100 = 20%
```

---

**Finding New Value After % Change:**

**Increase:**
```
New Value = Original √ó (1 + %Increase/100)
```

**Example:** Increase ‚Çπ500 by 20%
```
= 500 √ó (1 + 20/100)
= 500 √ó 1.2
= ‚Çπ600
```

**Decrease:**
```
New Value = Original √ó (1 - %Decrease/100)
```

**Example:** Decrease ‚Çπ800 by 20%
```
= 800 √ó (1 - 20/100)
= 800 √ó 0.8
= ‚Çπ640
```

---

#### **B. Successive Percentage Changes**

**Key Concept:** Two successive changes are NOT simply added!

**Formula for Two Changes:**
```
Net % = a + b + (ab/100)

where a and b are % changes (use - for decrease)
```

**Example:** Price increases 10%, then increases 20%
```
Net % = 10 + 20 + (10√ó20/100)
      = 30 + 2
      = 32%
```

**NOT 30%!**

---

**Example with Decrease:** Increase 20%, then decrease 10%
```
Net % = 20 + (-10) + (20√ó(-10)/100)
      = 20 - 10 - 2
      = 8% increase
```

---

**Alternative Method (Multipliers):**
```
New Value = Original √ó (1+a/100) √ó (1+b/100)
```

**Example:** ‚Çπ1000, +10% then +20%
```
= 1000 √ó 1.1 √ó 1.2
= 1000 √ó 1.32
= ‚Çπ1320 (32% increase)
```

---

#### **C. Percentage Points vs Percentage Change**

**CRITICAL DIFFERENCE:**

**Percentage Points:** Absolute difference
**Percentage Change:** Relative difference

**Example:** Rate changes from 20% to 25%

**Percentage Point Increase:** 25 - 20 = **5 percentage points**

**Percentage Increase:** 
```
= [(25-20)/20] √ó 100 = 25%
```

**Two different concepts!**

---

#### **D. Expressing One Quantity as % of Another**

**When comparing two quantities:**

**Example:** A = 80, B = 120

**A is what % of B?**
```
= (80/120) √ó 100 = 66.67%
```

**B is what % more than A?**
```
= [(120-80)/80] √ó 100 = 50% more
```

**A is what % less than B?**
```
= [(120-80)/120] √ó 100 = 33.33% less
```

**Note:** "More than" uses smaller as base, "less than" uses larger as base!

### **ADVANCED PROBLEM-SOLVING STRATEGIES**

#### **A. Percentage Change Reversal**

**Key Insight:** If something increases by x%, what % decrease brings it back?

**Formula:**
```
Required Decrease = [x/(100+x)] √ó 100
```

**Example:** Price increased 25%. What % decrease to restore?
```
= [25/(100+25)] √ó 100
= (25/125) √ó 100
= 20%
```

**Similarly, if decreased by x%:**
```
Required Increase = [x/(100-x)] √ó 100
```

**Example:** Price decreased 20%. What % increase to restore?
```
= [20/(100-20)] √ó 100
= (20/80) √ó 100
= 25%
```

**Pattern:** Restoring is NOT the same %!

---

#### **B. Population/Production Problems**

**Standard pattern:**
```
Final = Initial √ó (1 ¬± r/100)^n

+ for increase
- for decrease
n = number of periods
```

**Example:** Population 50,000, grows 10% annually for 2 years
```
= 50000 √ó (1.1)¬≤
= 50000 √ó 1.21
= 60,500
```

---

#### **C. Profit/Loss as Percentage**

**Formulas:**
```
Profit % = (Profit/Cost Price) √ó 100
Loss % = (Loss/Cost Price) √ó 100

Selling Price = CP + Profit = CP - Loss
```

**Example:** Bought for ‚Çπ500, sold for ‚Çπ600
```
Profit = 600 - 500 = ‚Çπ100
Profit % = (100/500) √ó 100 = 20%
```

---

**Finding SP from CP and Profit%:**
```
SP = CP √ó (1 + Profit%/100)
```

**Finding CP from SP and Profit%:**
```
CP = SP / (1 + Profit%/100)
```

---

#### **D. Percentage in Elections/Voting**

**Common structure:**
- Candidate A gets x%
- Candidate B gets remaining
- Margin of victory

**Example:** A gets 60%, B gets 40%, total 10,000 votes
```
A's votes = 60% of 10000 = 6000
B's votes = 40% of 10000 = 4000
Margin = 6000 - 4000 = 2000 votes
```

---

#### **E. Mixture and Alligation (Using Percentages)**

**When mixing two solutions:**

**Rule of Alligation:**
```
Ratio = (Higher % - Mean %) : (Mean % - Lower %)
```

**Example:** Mix 20% and 40% solutions to get 25%
```
Ratio = (40-25) : (25-20)
      = 15 : 5
      = 3 : 1
```

---

#### **F. Percentage in Time-Speed-Distance**

**Speed increase/decrease:**

**If speed increases by x%, time decreases by:**
```
[x/(100+x)] √ó 100%
```

**If speed decreases by x%, time increases by:**
```
[x/(100-x)] √ó 100%
```

**Example:** Speed increased 20%, time change?
```
Decrease = [20/120] √ó 100 = 16.67% decrease
```

## 2. FORMULAS & SHORTCUTS

### **MASTER FORMULA LIST**

#### **A. Basic Formulas**

**1. Percentage to Fraction:**

x% = x/100
```

**2. Finding x% of N:**
```
(x/100) √ó N

(N √ó 100) / x

#### **B. Increase/Decrease**

**5. Percentage Increase:**
```
[(New - Old) / Old] √ó 100
```

**6. Percentage Decrease:**
```
[(Old - New) / Old] √ó 100
```

**7. Value after Increase:**
```
Old √ó (1 + x/100)
```

**8. Value after Decrease:**
```
Old √ó (1 - x/100)
```

---

#### **C. Successive Changes**

**9. Two Successive % Changes:**
```
Net % = a + b + (ab/100)
```

**10. Three Successive Changes:**
```
Net = Original √ó (1+a/100)(1+b/100)(1+c/100)
```

---

#### **D. Special Cases**

**11. Restoring after Increase:**
```
Decrease needed = [x/(100+x)] √ó 100
```

**12. Restoring after Decrease:**
```
Increase needed = [x/(100-x)] √ó 100
```

**13. A is x% more than B:**
```
A = B(1 + x/100)
B = A/(1 + x/100)
```

**14. A is x% less than B:**
```
A = B(1 - x/100)
B = A/(1 - x/100)
```

### **TIME-SAVING SHORTCUTS**

#### **Shortcut 1: Finding 10%, 5%, 1%**

**10% = Move decimal one place left**
```
10% of 450 = 45
```

**5% = Half of 10%**
```
5% of 450 = 22.5
```

**1% = Move decimal two places left**
```
1% of 450 = 4.5
```

**Build other percentages:**
```
15% = 10% + 5% = 45 + 22.5 = 67.5
```

---

#### **Shortcut 2: Common Percentage Values (Memorize!)**

```
12.5% = 1/8 ‚Üí Divide by 8
25% = 1/4 ‚Üí Divide by 4
33.33% = 1/3 ‚Üí Divide by 3
50% = 1/2 ‚Üí Divide by 2
75% = 3/4 ‚Üí Three-fourths
```

**Example:** 25% of 80
```
= 1/4 of 80 = 20
```

**Faster than (25/100) √ó 80!**

---

#### **Shortcut 3: Successive % Changes (Quick)**

**For small numbers:**

**Example:** +10% then +20%
```
Step 1: Original = 100 (assume)
Step 2: After +10% = 110
Step 3: After +20% on 110 = 110 √ó 1.2 = 132
Net change = 32%
```

**Or use formula:** 10 + 20 + (10√ó20/100) = 32%

---

#### **Shortcut 4: Finding % Increase/Decrease**

**Quick mental pattern:**

**Increase from 50 to 60:**
```
Increase = 10, Base = 50
% = 10/50 = 1/5 = 20%
```

**Look for simple fractions!**

---

#### **Shortcut 5: Percentage More/Less**

**A is 25% more than B = 100**

**Quick:** A = 125

**B is what % less than A?**
```
Instead of calculation:
Difference = 25, New base = 125
25/125 = 1/5 = 20%
```

**Pattern:** More/less are NOT reciprocals!

---

#### **Shortcut 6: Reversal Percentages (Key Patterns)**

**Memorize these:**
```
+25% ‚Üí -20% to restore
+20% ‚Üí -16.67% to restore
+50% ‚Üí -33.33% to restore
+100% ‚Üí -50% to restore

-20% ‚Üí +25% to restore
-25% ‚Üí +33.33% to restore
-50% ‚Üí +100% to restore
```

---

#### **Shortcut 7: Election Margin**

**If A gets x% and B gets (100-x)%:**
```
Margin = Total √ó (2x - 100)/100
```

**Example:** A gets 60%, Total = 5000
```
Margin = 5000 √ó (120-100)/100 = 5000 √ó 0.2 = 1000
```

---

#### **Shortcut 8: Percentage from Ratio**

**If ratio A:B = 3:5**

**A's percentage:**
```
= [3/(3+5)] √ó 100 = (3/8) √ó 100 = 37.5%
```

**B's percentage:**
```
= (5/8) √ó 100 = 62.5%
```

---

#### **Shortcut 9: Quick Calculation for 15%, 35%, etc.**

**15% = 10% + 5%**
**35% = 10% √ó 3 + 5%**
**45% = 50% - 5%**

**Example:** 15% of 200
```
10% = 20
5% = 10
Total = 30
```

---

#### **Shortcut 10: Percentage Change in Products**

**If both dimensions change:**

**Length +20%, Width -10%**
```
Area change = 20 + (-10) + (20√ó(-10)/100)
            = 20 - 10 - 2
            = 8% increase
```

---

| Problem Type | Best Method | Time Saved |
