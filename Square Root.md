## 1. FOUNDATION TO ADVANCED COVERAGE

### **BASIC CONCEPTS (Foundation Level)**

#### **A. What is a Square Root?**

**Definition:** The square root of a number is a value that, when multiplied by itself, gives the original number.

```
If x² = n, then x = √n
```

**Example:**
- √25 = 5 (because 5 × 5 = 25)
- √64 = 8 (because 8 × 8 = 64)
- √144 = 12 (because 12 × 12 = 144)

**Real-world Example:** If a square field has area 100 m², its side length is √100 = 10 m.

#### **C. Properties of Square Roots**

**Property 1: Product Rule**
```
√(a × b) = √a × √b
```
**Example:** √36 = √(4 × 9) = √4 × √9 = 2 × 3 = 6

**Property 2: Quotient Rule**
```
√(a/b) = √a / √b
```
**Example:** √(25/4) = √25 / √4 = 5/2

**Property 3: Square Root of Square**
```
√(x²) = |x| = x (if x ≥ 0)
```
**Example:** √(7²) = 7

**Property 4: Square of Square Root**
```
(√x)² = x
```
**Example:** (√5)² = 5

**Property 5: Negative Numbers**
```
√(-n) is NOT a real number (for real number system)
```

### **INTERMEDIATE LEVEL TECHNIQUES**

#### **A. Long Division Method**

**For finding square root of non-perfect squares or verifying:**

**Example: Find √529**

**Step 1:** Pair digits from right: **5** | **29**

**Step 2:** Find largest square ≤ 5: 2² = 4
```
    2
   ___
  √529
   -4
   ---
   129
```

**Step 3:** Bring down next pair (29), making 129

**Step 4:** Double the quotient (2 × 2 = 4), write as 4_

**Step 5:** Find digit to fill blank: 4_ × _ ≤ 129
- Try 43 × 3 = 129 ✓

```
    23
   ___
  √529
   -4
   ---
   129
   -129
   ----
     0
```

**Answer: √529 = 23**

**For Exams:** Use only if answer options are close or for verification!

---

#### **B. Estimation Method (FASTEST for Exams)**

**Find approximate square root by sandwiching:**

**Example: √2000 ≈ ?**

**Step 1:** Find perfect squares around 2000
- 44² = 1936
- 45² = 2025

**Step 2:** 2000 is between 1936 and 2025
- So √2000 is between 44 and 45

**Step 3:** Closer to which?
- 2000 - 1936 = 64
- 2025 - 2000 = 25
- Closer to 2025, so closer to 45

**Answer: √2000 ≈ 44.7**

---

#### **C. Factorization Method**

**Best for simplifying square roots:**

**Example: Simplify √1800**

**Step 1:** Factorize
- 1800 = 2³ × 3² × 5² = 2² × 2 × 3² × 5²

**Step 2:** Take out perfect squares
- √1800 = √(2² × 3² × 5² × 2)
- = 2 × 3 × 5 × √2
- = 30√2

**Quick Pattern:**
```
√(a² × b) = a√b
```

### **ADVANCED PROBLEM-SOLVING STRATEGIES**

#### **A. Comparing Square Roots**

**Which is greater: √17 or 4?**

**Method 1: Square both sides**
- (√17)² vs 4²
- 17 vs 16
- 17 > 16, so √17 > 4

**Method 2: Approximate**
- 4² = 16, so 4 = √16
- √17 > √16, so √17 > 4

---

#### **B. Rationalization**

**Simplifying expressions with square roots in denominator:**

```
1/√a = √a/a
```

**Example:** 1/√5 = √5/5

**For binomials:**
```
1/(√a + √b) = (√a - √b)/(a - b)
```

**Example:** 1/(√5 + √3)
- Multiply by (√5 - √3)/(√5 - √3)
- = (√5 - √3)/(5 - 3)
- = (√5 - √3)/2

---

#### **C. Square Root Equations**

**Solving √x = a:**
```
Square both sides: x = a²
```

**Example:** √x = 7
- x = 49

**Solving √(x + 5) = 8:**
- x + 5 = 64
- x = 59

**Always verify** by substituting back!

---

#### **D. Complex Square Root Problems**

**Type 1: √(a + b√c)**

**Example: Simplify √(7 + 4√3)**

**Method:** Assume √(7 + 4√3) = √x + √y

Then: 7 + 4√3 = x + y + 2√(xy)

Comparing:
- x + y = 7
- 2√(xy) = 4√3, so xy = 12

Solve: x = 4, y = 3 (or vice versa)

**Answer: √(7 + 4√3) = 2 + √3**

---

#### **E. Square Root in Series**

**Finding patterns:**

```
√1 + √2 + √3 + ... + √n = ?
```

**No direct formula, but:**
- For perfect squares: Use actual values
- For approximations: Use estimation

## 2. FORMULAS & SHORTCUTS

### **MASTER FORMULA LIST**

#### **A. Basic Identities**

**1. Definition:**
```
√n × √n = n
(√n)² = n
√(n²) = n (for n ≥ 0)
```

**2. Product Rule:**
```
√(a × b) = √a × √b
```

**3. Quotient Rule:**
```
√(a/b) = √a/√b
```

**4. Power Rule:**
```
√(aⁿ) = (√a)ⁿ = a^(n/2)
```

**5. Nested Roots:**
```
√√a = ⁴√a = a^(1/4)
```

---

#### **B. Important Algebraic Identities**

**6. Sum/Difference:**
```
(√a + √b)² = a + b + 2√(ab)
(√a - √b)² = a + b - 2√(ab)
```

**7. Product of Conjugates:**
```
(√a + √b)(√a - √b) = a - b
```

**8. Triple Product:**
```
(√a + √b + √c)² = a + b + c + 2(√ab + √bc + √ca)
```

---

#### **C. Simplification Rules**

**9. Factoring Out:**
```
√(a²b) = a√b
√(a²b²c) = ab√c
```

**10. Rationalization:**
```
1/√a = √a/a

1/(√a + √b) = (√a - √b)/(a - b)

1/(√a - √b) = (√a + √b)/(a - b)
```

### **TIME-SAVING SHORTCUTS**

#### **Shortcut 1: Last Digit Method**

**Quick check if number is perfect square:**

Numbers ending in 2, 3, 7, 8 → **NEVER** perfect squares

**Instant elimination in MCQs!**

---

#### **Shortcut 2: Digital Root Method**

**Digital root** = Keep adding digits until single digit

**Perfect squares have digital roots:** 1, 4, 7, 9 ONLY

**Example:** Is 1234 a perfect square?
- Digital root: 1+2+3+4 = 10 → 1+0 = 1 ✓ (could be)
- But ends in 4, so could be
- Actually check: 35² = 1225, 36² = 1296 → NOT perfect square

**Use as quick filter, not definitive proof**

---

#### **Shortcut 3: Vedic Duplex Method**

**For 2-digit perfect squares near known values:**

**Example: 48² = ?**

Base = 50 (nearby multiple of 10)
- 48 = 50 - 2
- 48² = 50² - 2(50)(2) + 2²
- = 2500 - 200 + 4
- = 2304

**General:**
```
(a ± b)² = a² ± 2ab + b²
```

---

#### **Shortcut 4: Quick Squares (11-19)**

**Pattern:**
```
AB² where AB = (10 + B)
    = 1_B × B  |  B²
```

**Example: 13²**
- First part: 1 | 3×3 = 09 (write 1 in hundreds place)
- Second part: 3² = 09
- But add middle: 1 + (0+9 carry from 169) = 1
- Answer: 169

**Simpler:** Just memorize 11²-19²!

---

#### **Shortcut 5: Squares Ending in 5**

**Pattern:**
```
n5² = n(n+1) | 25
```

**Examples:**
- 15² = 1×2 | 25 = 225
- 25² = 2×3 | 25 = 625
- 35² = 3×4 | 25 = 1225
- 45² = 4×5 | 25 = 2025
- 85² = 8×9 | 25 = 7225

**Fastest method for _5 numbers!**

---

#### **Shortcut 6: Approximate Square Root (Babylonian)**

**For √N when you know nearby perfect square:**

```
√N ≈ (N + Perfect Square nearby) / (2 × √Perfect Square)
```

**Example: √50**
- Nearby: 49 (7²)
- √50 ≈ (50 + 49)/(2 × 7) = 99/14 ≈ 7.07

**Actual:** √50 ≈ 7.071 (very close!)

---

#### **Shortcut 7: Pattern Recognition**

**Recognize these instantly:**

```
√0.01 = 0.1
√0.04 = 0.2
√0.09 = 0.3
√0.16 = 0.4
√0.25 = 0.5
√0.36 = 0.6
√0.49 = 0.7
√0.64 = 0.8
√0.81 = 0.9

```

**Pattern:** √(n²/100) = n/10

---

#### **Shortcut 8: Sum of Squares Pattern**

```
1² + 2² + 3² + ... + n² = n(n+1)(2n+1)/6
```

**Example:** 1² + 2² + 3² + 4² + 5²
- = 5(6)(11)/6 = 330/6 = 55

---

#### **Shortcut 9: Difference of Squares**

```
a² - b² = (a+b)(a-b)
```

**Example:** 25² - 24²
- = (25+24)(25-24)
- = 49 × 1 = 49

**Use when finding differences!**

---

#### **Shortcut 10: Square Root of Product**

**If product has perfect square factors:**

**Example: √(16 × 25 × 9)**
- = √16 × √25 × √9
- = 4 × 5 × 3 = 60

**Faster than multiplying first!**
