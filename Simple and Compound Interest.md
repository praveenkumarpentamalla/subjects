# SIMPLE AND COMPOUND INTEREST: Complete Mastery Guide for Government Exams

## 1. FOUNDATION TO ADVANCED COVERAGE

### **BASIC CONCEPTS (Foundation Level)**

#### **A. What is Interest?**

**Definition:** Interest is the **cost of borrowing money** or the **reward for lending money**.

**Real-world Examples:**
- **Loan:** You borrow ‚Çπ10,000 from bank, pay back ‚Çπ11,000 ‚Üí Interest = ‚Çπ1,000
- **Savings:** You deposit ‚Çπ10,000, bank gives ‚Çπ10,500 after 1 year ‚Üí Interest = ‚Çπ500
- **Credit Card:** You spend ‚Çπ5,000, pay ‚Çπ5,100 next month ‚Üí Interest = ‚Çπ100

---

#### **B. Key Terms (Must Know)**

**1. Principal (P):**
- Original amount borrowed or invested
- Starting amount
- Example: Loan of ‚Çπ50,000 ‚Üí P = ‚Çπ50,000

**2. Rate of Interest (R):**
- Percentage charged per time period (usually per year)
- Example: 10% per annum ‚Üí R = 10%

**3. Time (T):**
- Duration for which money is borrowed/invested
- Usually in years
- Example: 3 years ‚Üí T = 3

**4. Interest (I):**
- Amount charged for borrowing/using money
- Extra money paid/received

**5. Amount (A):**
- Total money paid/received at the end
- Amount = Principal + Interest
- A = P + I

---

### **SIMPLE INTEREST (SI)**

#### **A. Basic Concept**

**Definition:** Interest calculated ONLY on the principal amount throughout the entire period.

**Formula:**
```
SI = (P √ó R √ó T) / 100
```

**Memory: "PRT by 100"**

---

#### **B. Understanding SI with Example**

**Example:** Principal = ‚Çπ1,000, Rate = 10% p.a., Time = 3 years

**Year-by-year:**
```
Year 1: Interest = 10% of 1000 = ‚Çπ100
Year 2: Interest = 10% of 1000 = ‚Çπ100 (still on original principal!)
Year 3: Interest = 10% of 1000 = ‚Çπ100
Total SI = ‚Çπ300
```

**Key Point:** Each year's interest is calculated on the ORIGINAL principal!

---

#### **C. Amount Formula for SI**

```
Amount = Principal + Simple Interest
A = P + SI
A = P + (PRT/100)
A = P(1 + RT/100)
```

**Example:** P = ‚Çπ10,000, R = 5%, T = 2 years
```
SI = (10000 √ó 5 √ó 2) / 100 = ‚Çπ1,000
Amount = 10,000 + 1,000 = ‚Çπ11,000
```

---

#### **D. Variations of SI Formula**

**Finding Principal:**
```
P = (SI √ó 100) / (R √ó T)
P = (A √ó 100) / (100 + RT)
```

**Finding Rate:**
```
R = (SI √ó 100) / (P √ó T)
```

**Finding Time:**
```
T = (SI √ó 100) / (P √ó R)
```

---

### **COMPOUND INTEREST (CI)**

#### **A. Basic Concept**

**Definition:** Interest calculated on the principal PLUS previously earned interest.

**Key Difference from SI:** Interest earns interest!

---

#### **B. Understanding CI with Example**

**Example:** Principal = ‚Çπ1,000, Rate = 10% p.a., Time = 3 years

**Year-by-year:**
```
Year 1: 
Amount = 1000 + 10% of 1000 = 1000 + 100 = ‚Çπ1,100
Interest = ‚Çπ100

Year 2:
Amount = 1100 + 10% of 1100 = 1100 + 110 = ‚Çπ1,210
Interest = ‚Çπ110 (on increased principal!)

Year 3:
Amount = 1210 + 10% of 1210 = 1210 + 121 = ‚Çπ1,331
Interest = ‚Çπ121

Total CI = 100 + 110 + 121 = ‚Çπ331
```

**Notice:** CI (‚Çπ331) > SI (‚Çπ300) for same P, R, T!

---

#### **C. CI Formula (Annual Compounding)**

```
Amount = P(1 + R/100)^T

Where:
P = Principal
R = Rate per annum
T = Time in years
```

**Compound Interest:**
```
CI = Amount - Principal
CI = P(1 + R/100)^T - P
CI = P[(1 + R/100)^T - 1]
```

---

#### **D. Different Compounding Frequencies**

**1. Annually (Once a year):**
```
A = P(1 + R/100)^T
```

**2. Half-Yearly (Twice a year):**
```
A = P(1 + R/200)^(2T)

R ‚Üí R/2 (divide rate by 2)
T ‚Üí 2T (multiply time by 2)
```

**3. Quarterly (4 times a year):**
```
A = P(1 + R/400)^(4T)

R ‚Üí R/4
T ‚Üí 4T
```

**4. Monthly (12 times a year):**
```
A = P(1 + R/1200)^(12T)

R ‚Üí R/12
T ‚Üí 12T
```

**Pattern:** 
- Divide rate by frequency
- Multiply time by frequency

---

#### **E. Special Cases in CI**

**1. Different Rates for Different Years:**

```
A = P(1 + R‚ÇÅ/100)(1 + R‚ÇÇ/100)(1 + R‚ÇÉ/100)...

Example: P = 10,000, Year 1: 10%, Year 2: 15%
A = 10000(1.1)(1.15) = 10000 √ó 1.265 = ‚Çπ12,650
```

---

**2. When Rate Increases/Decreases:**

**Population/Growth Problems:**
```
Final Value = Initial √ó (1 + R/100)^T  (increase)
Final Value = Initial √ó (1 - R/100)^T  (decrease)
```

**Example:** Population 10,000, increases 5% annually for 2 years
```
Final = 10000(1.05)¬≤ = 10000 √ó 1.1025 = 11,025
```

---

### **INTERMEDIATE LEVEL TECHNIQUES**

#### **A. Finding Time When Amount Doubles/Triples**

**For Simple Interest:**

**When amount doubles:**
```
Time = 100/R years
```

**Example:** At 5% SI, when does amount double?
```
T = 100/5 = 20 years
```

**When amount triples:**
```
Time = 200/R years
```

---

**For Compound Interest:**

**When amount doubles:**
```
(1 + R/100)^T = 2
```

**Common approximation (Rule of 72):**
```
Time ‚âà 72/R years
```

**Example:** At 8% CI, doubling time?
```
T ‚âà 72/8 = 9 years
```

---

#### **B. Difference Between CI and SI**

**For 2 Years:**
```
CI - SI = P(R/100)¬≤
```

**For 3 Years:**
```
CI - SI = P(R/100)¬≤(3 + R/100)
```

**Example:** P = 10,000, R = 10%, T = 2 years
```
SI = 10000 √ó 0.1 √ó 2 = ‚Çπ2,000
CI - SI = 10000 √ó (0.1)¬≤ = 10000 √ó 0.01 = ‚Çπ100
CI = 2000 + 100 = ‚Çπ2,100
```

**Quick formula saves time!**

---

#### **C. Installment Problems**

**Equal installments to pay off debt:**

**Formula:**
```
Each Installment = P √ó [(1 + R/100)^n √ó R] / [(1 + R/100)^n - 1]
```

**For exams:** Usually given as simple problems - calculate year by year.

---

### **ADVANCED PROBLEM-SOLVING STRATEGIES**

#### **A. Effective Rate for CI**

**When compounded more frequently:**

**Effective Annual Rate:**
```
Effective R = [(1 + r/n)^n - 1] √ó 100

where:
r = nominal rate
n = compounding frequency
```

**Example:** 12% compounded quarterly
```
Effective R = [(1 + 0.12/4)^4 - 1] √ó 100
            = [(1.03)^4 - 1] √ó 100
            = [1.1255 - 1] √ó 100
            = 12.55%
```

---

#### **B. Present Value and Future Value**

**Present Value (PV):**
- Amount needed today to get a certain amount in future

**Formula:**
```
PV = FV / (1 + R/100)^T
```

**Example:** What to invest today at 10% CI to get ‚Çπ12,100 after 2 years?
```
PV = 12100 / (1.1)¬≤
   = 12100 / 1.21
   = ‚Çπ10,000
```

---

#### **C. Continuous Compounding**

**Rarely in government exams, but good to know:**

```
A = Pe^(rt)

where e ‚âà 2.718
```

### **TIME-SAVING SHORTCUTS**

#### **Shortcut 1: Quick SI Calculation**

**For mental math:**
```
SI = P √ó R √ó T / 100
```

**Break it down:**
```
1% of P for T years = (P/100) √ó T
R% = R times that
```

**Example:** P = 5000, R = 6%, T = 3 years
```
1% of 5000 = 50
50 √ó 3 years = 150 per year
6% ‚Üí 150 √ó 6 = ‚Çπ900
```

---

#### **Shortcut 2: Doubling/Tripling (SI)**

**Instant formulas:**
```
Double: T = 100/R
Triple: T = 200/R
n times: T = 100(n-1)/R
```

**Example:** At 5% SI, to make amount 4 times?
```
T = 100(4-1)/5 = 300/5 = 60 years
```

---

#### **Shortcut 3: Rule of 72 (CI Doubling)**

**Quick approximation:**
```
Doubling Time ‚âà 72/R
```

**Example:** 12% CI
```
T ‚âà 72/12 = 6 years
```

**Also works for:**
- Rule of 69: More accurate (69/R)
- Rule of 70: Easy calculation (70/R)

---

#### **Shortcut 4: CI for 2 Years (Quick Method)**

**Instead of calculating year by year:**
```
A = P(1 + R/100)¬≤
  = P[1 + 2R/100 + (R/100)¬≤]
  = P[1 + 2R/100 + R¬≤/10000]
```

**Example:** P = 10,000, R = 10%, T = 2 years
```
A = 10000[1 + 0.2 + 0.01]
  = 10000 √ó 1.21 = ‚Çπ12,100
```

---

#### **Shortcut 5: CI - SI Difference (2 years)**

**Direct formula:**
```
Difference = P √ó (R/100)¬≤
```

**Example:** P = 8000, R = 5%
```
Diff = 8000 √ó (0.05)¬≤
     = 8000 √ó 0.0025
     = ‚Çπ20
```

**No need to calculate CI and SI separately!**

---

#### **Shortcut 6: Percentage Increase Pattern**

**For successive increases:**
```
10% increase then 20% increase ‚â† 30% increase!

Actual = (1.1 √ó 1.2 - 1) √ó 100 = 32%
```

**General:**
```

```

**When R and T are such that division is easy:**

**Example:** 10% half-yearly for 1 year
```
= 5% twice

```

#### **Shortcut 8: Power of Common Values**

**Memorize:**
```

---

#### **Shortcut 9: Finding Principal from Amount (CI)**

**When you know A, R, T:**
```
P = A / (1 + R/100)^T
```

**Example:** A = 12,100, R = 10%, T = 2 years
```

```

---

#### **Shortcut 10: Same SI for Different P, R, T**

**If SI is same:**
```
P‚ÇÅR‚ÇÅT‚ÇÅ = P‚ÇÇR‚ÇÇT‚ÇÇ

**Example:** If ‚Çπ1000 at 10% for 2 years gives same SI as ‚Çπ2000 at R% for 1 year:

## 5. COMMON MISTAKES & TRAPS

### **FREQUENT ERRORS STUDENTS MAKE**

#### **Error 1: Confusing SI and CI**

‚ùå **Wrong:** Using SI formula for CI problem

**Example:** "Amount after 2 years at 10% CI"
- Wrong: Using SI = PRT/100

‚úÖ **Correct:** Use A = P(1+R/100)¬≤

**Tip:** Keywords like "compounded," "population," "depreciation" ‚Üí CI

---

#### **Error 2: Not Converting Time for Half-Yearly/Quarterly**

‚ùå **Wrong:** Using T = 2 years directly for half-yearly

**Example:** 2 years half-yearly
- Wrong: T = 2

‚úÖ **Correct:** T = 2 √ó 2 = 4 half-years

**Also adjust R:**
- R = Annual/2 for half-yearly
- R = Annual/4 for quarterly

---

#### **Error 3: Forgetting to Find Interest (Finding Amount Only)**

‚ùå **Wrong:** Stopping at Amount when Interest is asked

**Example:** "Find CI on ‚Çπ10,000..."
- Wrong: Calculating A = 11,000 and stopping

‚úÖ **Correct:** CI = A - P = 11,000 - 10,000 = ‚Çπ1,000

---

#### **Error 4: Wrong Calculation in CI - SI Formula**

‚ùå **Wrong:** CI - SI = P √ó R¬≤ (missing /100¬≤)

‚úÖ **Correct:** CI - SI = P(R/100)¬≤

**Example:** P = 10000, R = 10%
- Wrong: 10000 √ó 100 = 1,000,000
- Correct: 10000 √ó 0.01 = ‚Çπ100

---

#### **Error 5: Using Doubling Formula Wrong**

‚ùå **Wrong:** Using T = 72/R for SI

‚úÖ **Correct:**
- SI: T = 100/R
- CI: T ‚âà 72/R (Rule of 72)

---

#### **Error 6: Negative Signs in Growth/Decline**

‚ùå **Wrong:** Population decrease 10% ‚Üí (1 + 10/100)

‚úÖ **Correct:** (1 - 10/100) = 0.9

**Use minus for decrease/depreciation!**

---

#### **Error 7: Not Simplifying Before Calculating**

‚ùå **Wrong:** (5000 √ó 6 √ó 3)/100 = 90000/100 (long division)

‚úÖ **Better:** Cancel zeros first
- = (50 √ó 6 √ó 3) = 900

### **TRICK QUESTIONS COMMONLY ASKED**

#### **Trick 1: The Half-Year Trap**

#### **Trick 2: The "Becomes" vs "Interest is" Trap**
