## 1. FOUNDATION TO ADVANCED COVERAGE

### **BASIC CONCEPTS & DEFINITIONS**

**What is a Decimal?**
A decimal is a way to represent fractions using a decimal point. The digits to the right of the decimal point represent parts of a whole.

**Place Value System:**
```
Units . Tenths Hundredths Thousandths Ten-thousandths
  5   .   7        3           8              2
```
- 5.7382 = 5 + 7/10 + 3/100 + 8/1000 + 2/10000

**Real-world example:** ‚Çπ45.75 means 45 rupees and 75 paise (75/100 of a rupee)

**Types of Decimals:**

1. **Terminating Decimals:** End after a finite number of digits
   - Example: 0.5, 0.25, 0.125

2. **Non-Terminating Repeating Decimals:** Digits repeat endlessly
   - Example: 0.333... (1/3), 0.142857142857... (1/7)

3. **Non-Terminating Non-Repeating:** Never end, never repeat
   - Example: œÄ = 3.14159... (not covered in competitive exams)

**Decimal to Fraction Conversion:**
- 0.5 = 5/10 = 1/2
- 0.25 = 25/100 = 1/4
- 0.75 = 75/100 = 3/4
- 0.125 = 125/1000 = 1/8

**Fraction to Decimal Conversion:**
Divide numerator by denominator
- 1/4 = 1 √∑ 4 = 0.25
- 3/8 = 3 √∑ 8 = 0.375

### **INTERMEDIATE LEVEL TECHNIQUES**

**Comparing Decimals:**
- Make the number of decimal places equal
- Compare: 0.5 vs 0.45
- Write as: 0.50 vs 0.45
- 0.50 > 0.45

**Ordering Decimals:**
Arrange from smallest to largest or vice versa
- 0.8, 0.75, 0.805, 0.9
- Standardize: 0.800, 0.750, 0.805, 0.900
- Order: 0.75 < 0.8 < 0.805 < 0.9

**Operations with Decimals:**

1. **Addition/Subtraction:**
   - Align decimal points
   - Add/subtract normally
   ```
     45.67
   + 12.30
   -------
     57.97
   ```

2. **Multiplication:**
   - Multiply ignoring decimals
   - Count total decimal places
   - Place decimal in answer
   ```
   2.5 √ó 3.2 = 25 √ó 32 = 800
   Total decimal places = 1 + 1 = 2
   Answer = 8.00 = 8
   ```

3. **Division:**
   - Make divisor a whole number
   - Adjust dividend accordingly
   - Divide normally
   ```
   12.5 √∑ 2.5 = 125 √∑ 25 = 5
   ```

## 2. ALL IMPORTANT FORMULAS & SHORTCUTS

### **BASIC FORMULAS**

**1. Decimal to Fraction:**
```
Decimal = Numerator/Denominator
Count decimal places = number of zeros in denominator
```
Example: 0.35 = 35/100 = 7/20 (2 decimal places = 100)

**2. Fraction to Decimal:**
```
Decimal = Numerator √∑ Denominator
```

**3. Pure Recurring Decimal to Fraction:**
```
0.ababab... = ab/99 (for 2-digit repeat)
0.abcabc... = abc/999 (for 3-digit repeat)
```

**4. Mixed Recurring Decimal:**
```
0.a(bc)ÃÖ where bc repeats
= [abc - a]/(99 √ó 10^(non-recurring places))
```

Example: 0.1666... = 0.1(6)ÃÖ
= (16 - 1)/90 = 15/90 = 1/6

**5. Adding/Subtracting Decimals:**
```
Align decimal points ‚Üí Add/Subtract
```

**6. Multiplying Decimals:**
```
Total decimal places in answer = Sum of decimal places in factors
```

**7. Dividing Decimals:**
```
Make divisor whole number by multiplying both by 10^n
```

## 5. COMMON MISTAKES & TRAPS

### **FREQUENT ERRORS STUDENTS MAKE**
