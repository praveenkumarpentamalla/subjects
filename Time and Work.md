# TIME AND WORK: Complete Mastery Guide for Government Exams

## 1. FOUNDATION TO ADVANCED COVERAGE

### **BASIC CONCEPTS (Foundation Level)**

**Core Definition:**
- **Work** = A task to be completed (always considered as 1 unit or 100%)
- **Time** = Days/hours needed to complete the work
- **Efficiency** = Work done per unit time

**Fundamental Principle:**
```
Work = Time √ó Efficiency
```

**Key Concept:** If A can do work in 10 days, then:
- A's 1 day work = 1/10
- A's efficiency = 10%

**Real-world Example:** If you can paint a room in 5 days working alone, you complete 1/5 (or 20%) of the room each day.

---

### **INTERMEDIATE LEVEL TECHNIQUES**

#### **A. Multiple Workers**

**Concept:** When people work together, their efficiencies ADD UP.

**Example:** 
- A completes work in 12 days ‚Üí A's 1 day work = 1/12
- B completes work in 15 days ‚Üí B's 1 day work = 1/15
- Together: (1/12 + 1/15) per day = (5+4)/60 = 9/60 = 3/20 per day
- Time together = 20/3 = 6‚Öî days

#### **B. Work and Wages**

**Principle:** Wages are distributed in the ratio of work done.

**Example:** A, B, C complete work in 6, 8, 12 days respectively.
- Efficiency ratio = 1/6 : 1/8 : 1/12 = 4:3:2
- If total wage = ‚Çπ1800, A gets = (4/9) √ó 1800 = ‚Çπ800

#### **C. Alternate Working Days**

**Pattern:** When workers alternate days, calculate work done in complete cycles.

---

### **ADVANCED PROBLEM-SOLVING STRATEGIES**

#### **A. Chain Work (Part by Part)**

When workers complete different portions of work:
- Calculate each person's contribution separately
- Add all contributions to equal 1 (complete work)

#### **B. Negative Work (Damage/Leakage)**

When someone undoes work (leakage in tank, error in work):
- Treat as negative efficiency
- Subtract from positive work

**Example:** Pipe A fills tank in 6 hours, Pipe B empties in 8 hours.
- Combined: 1/6 - 1/8 = (4-3)/24 = 1/24 per hour
- Together they fill in 24 hours

#### **C. Group Variations**

Men-Women-Children problems with different efficiencies:
- Always convert to one standard unit (usually "man-days")
- Use ratio of efficiencies

**Standard Ratios (commonly used):**
- 1 Man = 2 Women = 3 Children (typical assumption)

#### **D. Work Efficiency with Time Changes**

When working hours change daily:
- Calculate actual "work-hours"
- Efficiency per hour remains constant

## 2. FORMULAS & SHORTCUTS

### **MASTER FORMULA LIST**

#### **A. Basic Formulas**

1. **Work = Time √ó Efficiency**

2. **If A completes work in 'n' days:**
   - A's 1 day work = 1/n
   - A's efficiency = (100/n)%

3. **Combined Work (A and B together):**
   ```
   Time together = (A √ó B)/(A + B)
   ```
   Where A and B are individual days

4. **Three workers together:**
   ```
   Time = (A √ó B √ó C)/(AB + BC + CA)
   ```

5. **If A and B together complete in T days, and A alone in A days:**
   ```
   B's time = (A √ó T)/(A - T)
   ```

#### **B. Advanced Formulas**

6. **M‚ÇÅD‚ÇÅW‚ÇÇ = M‚ÇÇD‚ÇÇW‚ÇÅ**
   - M = Number of men
   - D = Number of days
   - W = Work done
   
   Extended: **M‚ÇÅD‚ÇÅH‚ÇÅ/W‚ÇÅ = M‚ÇÇD‚ÇÇH‚ÇÇ/W‚ÇÇ** (H = hours per day)

7. **Efficiency and Time (Inverse Relation):**
   ```
   E‚ÇÅ/E‚ÇÇ = T‚ÇÇ/T‚ÇÅ
   ```

8. **Remaining Work Formula:**
   ```
   Remaining work = 1 - (days worked/total days needed)
   ```

9. **Work and Wages:**
   ```
   Wage ratio = Efficiency ratio = Time ratio (inverse)
   ```

10. **Alternate Day Work:**
    ```

    ```

### **TIME-SAVING SHORTCUTS**

#### **Shortcut 1: LCM Method (FASTEST for exam)**

**When to use:** Any combined work problem

**Method:**
1. Take LCM of all individual days = Total work
2. Calculate efficiency = Total work/Individual days
3. Combined efficiency = Sum of individual efficiencies
4. Time = Total work/Combined efficiency

**Example:** A=12 days, B=15 days, find combined time
- LCM(12,15) = 60 units (Total work)
- A's efficiency = 60/12 = 5 units/day
- B's efficiency = 60/15 = 4 units/day
- Together = 5+4 = 9 units/day
- Time = 60/9 = 6‚Öî days

**Why it's best:** No fractions until the end!

---

#### **Shortcut 2: Direct Formula (Under 10 seconds)**

**For 2 workers:**
```
Time together = (Product)/(Sum) = (A√óB)/(A+B)
```

**Example:** A=10 days, B=15 days
- Together = (10√ó15)/(10+15) = 150/25 = 6 days

---

#### **Shortcut 3: Percentage Method**

**When to use:** Problems with percentages or fractional work

**Method:**
- Total work = 100%
- Calculate daily efficiency as percentage
- Remaining work = 100% - work done%

**Example:** A does 20% per day, B does 25% per day
- Together = 45% per day
- Time = 100/45 = 2.22 days

---

#### **Shortcut 4: Ratio Method for Wages**

**Quick Ratio Conversion:**
If A takes 6 days, B takes 8 days, C takes 12 days:
- Time ratio = 6:8:12
- Efficiency ratio = 1/6:1/8:1/12 = **4:3:2** (multiply by LCM=24)
- Wage ratio = 4:3:2

---

#### **Shortcut 5: One-Third Rule**

If A, B, C can do work individually in a, b, c days:
```
Together time ‚âà a/3 (when a‚âàb‚âàc)
```

---

### **ALTERNATIVE METHODS**

**Problem Type: A and B together in 12 days, A alone in 20 days, find B?**

**Method 1 (Traditional):**
- 1/20 + 1/B = 1/12
- 1/B = 1/12 - 1/20 = (5-3)/60 = 2/60 = 1/30
- B = 30 days

**Method 2 (Direct Formula):**
```
B = (A √ó T)/(A - T) = (20 √ó 12)/(20-12) = 240/8 = 30 days
```

**Method 3 (LCM - Fastest):**
- LCM(12,20) = 60
- Combined efficiency = 60/12 = 5
- A's efficiency = 60/20 = 3
- B's efficiency = 5-3 = 2
- B's time = 60/2 = 30 days

---

### **WHEN TO USE WHICH METHOD**

| Problem Type | Best Method | Time Taken |

## 5. COMMON MISTAKES & TRAPS

### **FREQUENT ERRORS STUDENTS MAKE**

#### **Error 1: Forgetting to Find TOTAL Work First**

‚ùå **Wrong:** "A does 1/10 per day, works 3 days, so remaining = 1 - 3/10"
‚úÖ **Correct:** First ensure what "1" represents (complete work)

**Trap Question:** "A completes 3/5 of work in 12 days. How many days for full work?"
- Trap: Thinking answer is 12 days
- Correct: 12 days for 3/5 ‚Üí Full work = 12√ó(5/3) = 20 days

---

#### **Error 2: Adding Times Instead of Efficiencies**

‚ùå **Wrong:** "A = 10 days, B = 15 days ‚Üí Together = 10+15 = 25 days"
‚úÖ **Correct:** Add efficiencies (1/10 + 1/15), not times!

**Remember:** Times ADD only when people work SEPARATELY, not together!

---

#### **Error 3: Wrong Inverse Relationship**

‚ùå **Wrong:** "A is 50% more efficient ‚Üí takes 50% more time"
‚úÖ **Correct:** 50% more efficiency ‚Üí takes 2/3 time (inverse!)

**Pattern:**
- 100% more efficiency (double) ‚Üí Half time
- 50% more efficiency (3/2) ‚Üí 2/3 time
- 200% more efficiency (triple) ‚Üí 1/3 time

---

#### **Error 4: Forgetting Negative Work**

**Trap:** "Pipe fills in 6 hours, leak empties in 8 hours, both open?"
‚ùå **Wrong:** (1/6 + 1/8) [Adding both]
‚úÖ **Correct:** (1/6 - 1/8) [Leak is negative]

---

#### **Error 5: Mixing Up Ratios**

**Trap:** "Wages in ratio 3:4, what is A's share in ‚Çπ700?"
‚ùå **Wrong:** A gets ‚Çπ300
‚úÖ **Correct:** A gets (3/7)√ó700 = ‚Çπ300

Common mistake: Forgetting to use total ratio parts (3+4=7)

---

#### **Error 6: Calculation Errors with Fractions**

### **TRICK QUESTIONS COMMONLY ASKED**

#### **Trick 1: The Efficiency Deception**

**Question:** "A is 25% less efficient than B. If B takes 20 days, how many days will A take?"

**Trap:** Thinking A takes 20 - 25% = 15 days
**Reality:** 
- B = 100%, A = 75%
- Ratio B:A = 100:75 = 4:3
- Time ratio = 3:4 (inverse)
- A takes = 20√ó(4/3) = 26.67 days

**Rule:** "Less efficient" ‚Üí More time (more than 20!)

---

#### **Trick 2: The Partial Work Trap**

**Question:** "A completes 2/3 of work in 8 days. B completes 1/2 of work in 6 days. Together?"

**Trap:** Using 8 and 6 directly
**Correct Method:**
- A's full work = 8√ó(3/2) = 12 days
- B's full work = 6√ó2 = 12 days
- Together = (12√ó12)/(12+12) = 6 days

---

#### **Trick 3: The "Per Day" Confusion**

**Question:** "A does work in 10 days working 6 hours/day. How many days if working 8 hours/day?"

**Trap:** Thinking more hours = more days
**Reality:** More hours = fewer days!
- Total work = 10√ó6 = 60 hours
- At 8 hrs/day = 60/8 = 7.5 days

---

#### **Trick 4: The Alternating Trap**

**Question:** "A and B work on alternate days starting with A. A=10 days, B=15 days. Total days?"

**Trap:** Using simple together formula
**Must:** Calculate 2-day cycles properly
- 2-day work = 1/10 + 1/15 = 1/6
- Cycles = 6 (but last cycle may be incomplete)
- Careful calculation needed!

---

#### **Trick 5: The Wage Distribution Confusion**

**Question:** "A, B, C get wages in ratio 2:3:5. If B gets ‚Çπ60, what does C get?"

**Trap:** C/B = 5/3, so C = 60√ó(5/3) = ‚Çπ100
**Wait!** Reread - is ratio of wages or work or time?
- If work ratio ‚Üí wages same as work ratio
- If time ratio ‚Üí wages inverse of time ratio!

### **HOW TO AVOID CALCULATION MISTAKES**

#### **Strategy 1: Use LCM to Avoid Fractions**

Instead of: 1/12 + 1/15 = ?
Do: LCM = 60, so 5 + 4 = 9, answer = 9/60 = 3/20

#### **Strategy 2: Double-Check Inverse Relations**

- "More efficient" ‚Üí Less time
- "More workers" ‚Üí Less time
- "More hours/day" ‚Üí Less days

#### **Strategy 3: Verify with Approximate Values**

#### **Strategy 4: Write Down Given Information Clearly**

- A = ? days
- B = ? days

- Find = ?

This prevents mixing up what to find!

#### **Strategy 5: Cross-Verify Final Answer**
